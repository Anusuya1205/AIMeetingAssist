# 🤖 AI-Powered Meeting Assistant

## 📌 Project Description
AI-Powered Meeting Assistant is a .NET Web API application that generates automated Minutes of Meeting (MoM) using AI. It converts meeting inputs into structured summaries, improving productivity and reducing manual effort.

---

## 🚀 Features
- AI-based meeting summary generation  
- REST API using ASP.NET Core  
- Swagger integration for API testing  
- Scalable backend structure  

---

## 🛠️ Tech Stack
- ASP.NET Core Web API  
- C#  
- AI Integration (LLM)  
- Swagger  
- Git & GitHub  

---
